## 🚀 Featured Projects

### 🌦️ [Weather Prediction Bot](https://github.com/vaibhav-mi/weather-prediction-bot)

A weather forecasting app powered by a TensorFlow/Keras image-classification model. A Flask + Flask-CORS API serves predictions, with Selenium and BeautifulSoup handling live data scraping and APScheduler running the refresh jobs, while a Vite-powered JS frontend calls `/api/predict` for real-time results.

`Python` `TensorFlow` `scikit-learn` `OpenCV` `Flask` `Vite`

---

### 🗂️ [Devfolio](https://github.com/vaibhav-mi/Devfolio)

A full-stack Django CRUD dashboard for managing an entire dev career in one place — skills, projects, work history, and certifications — complete with auth, a Chart.js-powered analytics dashboard, and Bootstrap 5 + Crispy Forms styling.

`Python` `Django` `SQLite` `Bootstrap` `Chart.js`
